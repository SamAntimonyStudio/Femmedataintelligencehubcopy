Design System

Brand & Visual System

All screens must strictly follow this system. No deviations. Every component, colour, and typographic choice defined here.

Typography
  • Cormorant Garamond — All headings, dashboard titles, KPI values, screen names
  • DM Sans — All body copy, labels, descriptions, table content
  • DM Mono — All metric labels, data tags, section labels, code values, status pills
Colour Palette
  • #1b443a — Forest Green · Primary brand, nav, headers
  • #ff99b9 — Pink · Accents, active states, AI highlights
  • #e2745d — Terracotta · Alerts, warnings, secondary accents
  • #e6d1c4 — Buff · Backgrounds, subtle cards, borders
  • #000000 / #ffffff — Black / White · Nav, type, surfaces
Spacing & Radius
  • Canvas: 1680 × 1050px — fixed desktop
  • Nav sidebar: 80px wide · fixed left
  • Grid: 12-column at 1680px, 24px gutter
  • Card radius: 16px standard, 10px compact
  • Card padding: 24px standard, 20px compact
  • Gap between cards: 16px
Global Layout — Every Screen
  • Left nav (80px): FC logo top · icon nav with labels · user avatar bottom · collapse state available
  • Top header bar: Dashboard title + breadcrumb · Date range picker (Last 7D / 30D / 90D / Custom / vs Prior Period) · Export · Settings icon
  • Sub-nav tabs: Overview · By Channel · By Product · Diagnostics · AI Insights (where relevant)
  • KPI row: 5–6 cards spanning full width · each with value, label, delta %, sparkline, period
  • Strategic action strip: Silver Bullet · Low-Hanging Fruit · Long-Term Task — 3-card horizontal strip below KPI row on every screen
  • Right column (30%): AI insight panel on all screens
  • Floating AI chat: Bottom-left corner · global across all screens
Global Components — Every Screen
  • Femme Connection logo: Top-left of nav sidebar — always visible
  • User profile avatar: Bottom of nav sidebar
  • ⓪ Info/context panel: On every card — click to reveal data source, methodology, and AI context
  • Date range picker: Global — propagates across all modules when changed
  • Alert banners: Top of relevant screens — Red (critical), Amber (opportunity), Green (positive signal)
  • AI insight cards: Pre-filled with realistic strategic copy — no placeholder text
  • Drill-down drawer: Triggered from priority action cards — slides in from right
Real Content Rules — Non-Negotiable

No placeholder text anywhere. All product names, metric values, competitor names, and copy must be real. Use these product names: Daisy Floral Maxi Dress · Evelina Sequin Maxi Dress · Jania Solid Jogger Pants · Morocco Floral Maxi Dress · Ronika Floral Print Dress · Kalinda Abstract Dress · Luxe Linen Wrap Top · Marrakesh Floral Maxi · Royce Embroidered Midi · Agatha Floral Shirt Dress.
